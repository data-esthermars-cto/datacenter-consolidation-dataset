# Data Center Consolidation Planning Dataset

A collection of 13 structured files representing pre-migration due diligence documentation for a multi-site data center consolidation project. The dataset covers server infrastructure, application portfolio, vendor contracts, infrastructure monitoring, incident management, security assessments, network topology, software licensing, change management, business continuity, operating budget, facility specifications, and storage health diagnostics.

## Contents

| File | Format | Description |
|------|--------|-------------|
| server_asset_register.xlsx | XLSX | Hardware inventory across three data center sites |
| application_portfolio_catalog.xlsx | XLSX | Application registry with hosting and dependency mappings |
| vendor_contract_register.xlsx | XLSX | Vendor agreements, contract terms, and pricing |
| infrastructure_monitoring_q4_2024.csv | CSV | Server performance telemetry for Q4 2024 |
| incident_management_log_2024.csv | CSV | IT incident records for calendar year 2024 |
| quarterly_security_assessment_q4_2024.pdf | PDF | Vulnerability assessment and security findings |
| network_topology_and_vlan_assignments.csv | CSV | Network segmentation and VLAN configuration |
| software_license_compliance_audit.xlsx | XLSX | Software deployment versus entitlement reconciliation |
| change_request_log_h2_2024.csv | CSV | Change management records for H2 2024 |
| business_continuity_plan_2024.pdf | PDF | Disaster recovery and business continuity documentation |
| it_operating_budget_fy2025.xlsx | XLSX | IT infrastructure operating budget |
| datacenter_facility_specifications.pdf | PDF | Physical facility specifications and constraints |
| storage_health_assessment_q4_2024.xlsx | XLSX | Storage array health and capacity diagnostics |
